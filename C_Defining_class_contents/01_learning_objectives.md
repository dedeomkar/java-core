## 1. Course Overview

### 1.1 Key Topics Covered

#### Instance and Static Fields
- **Declaration**: Declaring instance and static fields within classes.
- **Initialization**: Understanding default initialization rules.
- **Access**: Mechanisms and rules for accessing these fields.

#### Instance and Static Methods
- **Declaration & Signature**: Defining formal parameter lists and throws clauses.
- **Implementation**: Writing method bodies for both instance and static contexts.
- **Receiver Parameter**: Understanding the implicit `this` value that acts as the receiver parameter for instance methods.

#### Argument Passing and Method Signatures
- **Argument Passing**: Understanding Java's pass-by-value rules for arguments.
- **Varargs**: Handling varying numbers of arguments using variable-length argument lists.
- **Polymorphism**: Differentiating and resolving overloaded (compile-time) and overridden (run-time) methods.

#### Java Records
- **Data Carriers**: Introduction to defining and using record types as immutable data aggregates.
